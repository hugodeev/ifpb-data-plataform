# IFPB Data Platform - Observatório de Dados Abertos

## Descrição do Projeto

O **IFPB Data Platform** é uma plataforma web interativa desenvolvida como projeto de Iniciação Científica PIBIC/CNPq no IFPB Campus Guarabira. A plataforma tem como objetivo principal promover a **transparência ativa** e facilitar o **acesso democrático** aos dados institucionais do Instituto Federal da Paraíba.

## Objetivo Principal

Desenvolver um observatório digital que integra, processa e visualiza dados públicos do IFPB de forma intuitiva e acessível, transformando informações técnicas em conhecimento compreensível para diversos públicos (estudantes, professores, gestores e sociedade em geral).

## 🔧 Metodologia e Fluxo de Trabalho

O projeto segue uma abordagem completa de **ciência de dados aplicada**:

1. **Coleta de Dados**: Extração de dados públicos de fontes oficiais do IFPB
2. **Processamento**: Limpeza e organização utilizando Python e Pandas no Google Colab
3. **Análise**: Transformação de dados brutos em informações significativas
4. **Visualização**: Criação de gráficos interativos com Chart.js e Vega-Lite
5. **Interface**: Desenvolvimento web com HTML5, CSS3 e JavaScript
6. **Publicação**: Disponibilização em plataforma web responsiva

## Arquitetura Técnica

- **Backend de Dados**: Processamento em Python/Pandas (Google Colab)
- **Frontend**: HTML5 semântico, CSS3 moderno, JavaScript ES6+
- **Visualizações**: Chart.js para gráficos simples, Vega-Lite para visualizações complexas
- **Design**: Interface responsiva (mobile-first), sistema de design consistente

## Dados Abrangidos

A plataforma integra múltiplas dimensões dos dados institucionais:

- **Estruturais**: Informações sobre os 18 campi do IFPB
- **Acadêmicos**: Cursos, matrículas, corpo docente e discente
- **Financeiros**: Orçamento, receitas, despesas e investimentos
- **Gestão**: Indicadores de desempenho e eficiência
- **Pesquisa**: Dados primários coletados junto à comunidade acadêmica

## Impacto e Inovação

O projeto representa uma **inovação na transparência pública** porque:

1. **Integração**: Une dados fragmentados em uma única interface
2. **Interatividade**: Permite exploração ativa dos dados pelo usuário
3. **Acessibilidade**: Traduz dados técnicos em visualizações compreensíveis
4. **Formação**: Desenvolve competências em ciência de dados nos estudantes
5. **Sustentabilidade**: Utiliza tecnologias open-source e licença aberta

## Contexto Educacional

Desenvolvido por estudante do **Curso Técnico em Informática Integrado ao Ensino Médio**, o projeto demonstra:

- Aplicação prática do conhecimento técnico adquirido
- Integração entre formação técnica e pesquisa científica
- Contribuição concreta para a instituição de ensino
